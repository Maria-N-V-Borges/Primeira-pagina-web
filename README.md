# Minha primeira página web 🌸
Projeto iniciado com HTML5 e CSS3 na disciplina de Desenvolvimento Web, evoluído com JavaScript, Dark Mode, menu hambúrguer e persistência de tema (localStorage).

👉 Veja online: [Clique aqui](https://maria-n-v-borges.github.io/Primeira-pagina-web/)

---

## 📄 Páginas do projeto

Início | Blog | Projetos | Tabela | Formulário | Lâmpada (JS) | Dark Mode | Sobre | Contato

## 🧠 Tecnologias e conceitos
- **HTML5**: estrutura semântica, formulários, tabelas
- **CSS3**: flexbox, grid, gradientes, responsividade em andamento, menu com hover
- **JavaScript**: DOM, eventos, funções, arrow functions, operador ternário, localStorage
- **Destaque**: lâmpada interativa com brilho e clique

## 💡 Dificuldades superadas
- Separação de CSS em múltiplos arquivos
- Persistência do dark mode entre páginas
- HTML puro não envia dados de formulário (backend futuro)
