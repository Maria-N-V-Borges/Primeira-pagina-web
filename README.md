# Minha primeira página web 🌸

Projeto iniciado com HTML5 e CSS3 na disciplina de Desenvolvimento Web, evoluído com JavaScript, Dark Mode, menu hambúrguer, persistência de tema (localStorage) e **formulário funcional com integração ao Formspree**.

👉 **Veja online:** [Clique aqui](https://maria-n-v-borges.github.io/Primeira-pagina-web/) <br>
👉 **Formulário:** [Clique aqui](https://maria-n-v-borges.github.io/Primeira-pagina-web/formulario.html)

---

## 📄 Páginas do projeto

🏠 Início | ✍️ Blog | 🚀 Projetos | 📊 Tabela | 📝 Formulário | 💡 Lâmpada (JS) | 🌙 Dark Mode | 👩‍💻 Sobre | 📬 Contato

---

## 🧠 Tecnologias e conceitos

- **HTML5**: estrutura semântica, formulários, tabelas, inputs variados (texto, telefone, data, checkbox, radio, select, textarea)
- **CSS3**: flexbox, grid, gradientes, responsividade (em andamento), menu com hover, dark mode com transição suave
- **JavaScript**: DOM, eventos, funções, arrow functions, operador ternário, localStorage (persistência do tema), menu hambúrguer
- **Integração com API**: Formspree para envio de formulários (backend sem servidor próprio)

### ✨ Destaques do projeto
- **Lâmpada interativa** com efeito de brilho e clique (manipulação de DOM + CSS dinâmico)
- **Formulário funcional** que envia dados para e-mail em tempo real via Formspree
- **Dark Mode persistente** entre todas as páginas usando localStorage

## 💡 Dificuldades superadas

| Desafio | Solução |
|---------|---------|
| Separar CSS em múltiplos arquivos | Organização em pastas (`css/style.css`, `css/blog.css`, etc.) |
| Persistir Dark Mode entre páginas | Uso de `localStorage` + JavaScript |
| Formulário não enviava dados | Integração com Formspree (backend como serviço) |
| GitHub Pages não roda PHP | Adoção de soluções externas para formulários |
